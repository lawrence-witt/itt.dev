# itt.dev

itt.dev is my personal web developer portfolio, collating and showcasing the progress I have made so far during my programming journey. Visit the site for contact links, a full list of technical proficiencies, and more detailed explanations of the development process behind each of my projects.

[Live Site](https://itt.dev)

## Features

- [x] PHP back end including a simple yet flexible custom templating class
- [x] WCAG compliant, IE11 compatible HTML/CSS styled using a clean, modular design
- [x] Site data stored in discrete JSON files to simplify the adding and updating of content
- [x] Light and dark mode themes, chosen based on system preference and user selection
